### サービス概要
SpotSnapを一言で言うと「場所共有・検索に特化したSNSアプリ」です。

例えば、投稿にはその場所の写真と位置情報を必須で載せるような仕様となっており、投稿されるコンテンツを制限しています。

また、マップから場所検索が可能であり、キーワードやサジェストによる検索が可能となっています。

基本的な設計はインスタグラムを意識しており、そこから「場所」の共有を意識した設計製造を行なっています。

### コンセプト
このアプリを作成しようと思った背景としては、「インスタって、場所を探す時によく使われているけど結構使いにくいな」と思ったことがきっかけです。

私自身、旅行に行き写真を撮るのが好きなので、インスタグラムを使って旅行先の場所を探したり、旅行先の写真を投稿したりすることが多いのですが、インスタグラムで場所を探す時に不便さを感じることも多かったので、その使いにくさを解消するようなアプリを作りたいと思いました。

このアプリの魅力を一言で言うと「検索のしやすさ」です。インスタで場所を検索しようとすると「#東京サウナ」のようなタグに紐づく場所しか表示されないですが、このアプリではGoogleデータとの紐づけから場所を探すため、キーワードやサジェストによる検索が可能となっています。

今後はさらにGoogle評価によるマップ表示の絞り込み機能、場所ごとに総いいね数/投稿数から「人気」「穴場」などのステータス機能などを追加していく予定です。

### ユーザー層
「旅行に行くから情報収集したい！」
旅行先の場所で、Googleのようにキーワードなど広い条件で場所を検索したい方

「お気に入りの場所を共有したい！」
よく行く場所、お気に入りの場所を思い出と共に他の人にも共有したいという方

「暇なときに旅行雑誌のように見たい！」
暇なときに場所を探して、出かけるときのためにストックしておきたい方

### 実装機能
- アカウント登録機能
- ログイン機能
- 管理者機能（ユーザー削除）
- メール認証機能
- パスワードリセット機能
- ホーム
  - いいねごとの投稿一覧
  - 自分のした投稿一覧
  - いいね機能
- 投稿画面
  - プレビュー機能
  - 画像比選択機能
- ユーザー検索画面
- マップ画面
  - 投稿の位置情報表示機能
  - オートコンプリート機能
  - キーワード検索機能
- 場所ごとの投稿詳細画面
  - 住所/投稿数/いいね数の表示
- プロフィール画面
  - いいねごとの投稿一覧
  - 自分のした投稿一覧

 ### 実装予定の機能
- いいね数と投稿数を用いたステータス機能
  - いいね数も稿数数も多い投稿→人気スポット
  - いいね数が多く、投稿数の少ない投稿→穴場スポット
- 場所詳細画面に投稿画像とは別にGoogleMapの写真を表示させる機能
- マップ画面の表示ウィンドゥと場所詳細画面に表示する情報を増やす
  - Googleのレビュー数
  - Googleの評価（星）
  - 営業時間
- 現在地をマップに表示させる機能
  - 例：評価数が高い順（条件）　×　3 (表示数)
